## О проекте

Проект представляет из себя оверлей с таймерами/секундомерами.
Каждый секундомер/таймер в списке имеет у себя элементы управления. Стоп и стар, настройки и удаление
У каждого таймера/секундомера есть модальное окно настроек
под под списком кнопка добавления таймера/секундомера
есть 4 вида таймера/секундомера

- секундомер
- таймер
- таймер до даты-время
- цикличный таймер (отсчитывает время назад, после достижения 0 через 10 секунд сбрасывается на duration)
- цикличный таймер от времени (отсчитывает время до указанного времени суток, после достижения 0 циклично перезапускается)

## Стек технологий

- HTML5
- CSS3
- JavaScript
- EmberJs
- Electron
- pods структура

## Директории

`/pods` — папка с описанием контроллеров, роутов и шаблонов - По сути это описание страницы
`/components` — папка компонентами

Остальные директории вспомогательные, создаются при необходимости разработчиком

## Запуск проекта

`yarn start` — запускает EmberJs сервер
`yarn start-electron-app` — запускает дев сервер приложения Electron с предварительно запущенным EmberJs сервером
`yarn electron-pack` — запускает продакшен сборку приложения Electron
